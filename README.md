# Fishing Horizons: Expanded

A SMAPI (C#) companion mod for [Fishing Horizons](https://github.com/Fishing-Horizons/Fishing-Horizons) that adds new fishing mechanics to Stardew Valley.

Mechanics are built as independent **modules** that can be toggled in the config (GMCM supported).

## Modules

| Module | Status | Description |
|--------|--------|-------------|
| **Fisherman's Journal** | 🚧 In development | An in-game book that tracks every fish you've caught: catch stats, max size, sold/gifted achievements, and where & when to catch each fish. Works with vanilla fish and fish from any installed mod. |

## Requirements

- Stardew Valley 1.6+
- SMAPI 4.0+
- (Optional) Generic Mod Config Menu

## Building

The project uses [Pathoschild.Stardew.ModBuildConfig](https://www.nuget.org/packages/Pathoschild.Stardew.ModBuildConfig), which automatically finds your game folder. Open the solution and build — the mod is copied to your `Mods` folder automatically.
